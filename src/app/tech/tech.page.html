The Following technologies will be used
  • Angular / Ionic

    • The whole application will be build with Ionic which is build on the top of Angular.
  • SQLite3 or MongoDB

    • The plan is to have a cloud database in the future and share shopping list across multiple clients. However, a standalone database is an essential for a stand alone application configuration.
  • Robot Framework / Selenium / Python3

    • A side project for automation/acceptance system testing of the application.
